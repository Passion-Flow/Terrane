# Onboarding 四幕剧分幕剧本（PRD 4.11.1 落地 spec）

前置：部署初始化向导完成（License 激活→超管改密→邮件→Branding）。四幕在首个普通用户首次登录触发；每幕可跳过；埋点事件写 product_events（act_started/act_completed/act_skipped + aha 埋点）。

## 第一幕「喂给它」（目标 0-3 分钟，aha₁ = 看着知识长出来）
1. 欢迎层（spotlight 全屏）："给 Terrane 一些你的资料，看它如何长成你的第二大脑"——三入口卡：拖拽文件 / 装剪藏插件 / 接一个连接器（推荐顺序按检测：有 Obsidian vault 路径→优先文件夹）
2. 用户投放首批源 → 自动切到库工作台双栏，右栏摄入活动流实时滚动（解析→分块→图谱节点逐个长出）
3. 摄入完成 toast："已编译 N 个知识页"→ **埋点 aha1_kb_growing**（右栏首次 SSE 渲染新页即触发）
- 降级：用户没有现成文件 → 提供"示例库"一键导入（出厂 3 篇示例文档，不算 aha）

## 第二幕「问它 + 看见大脑」（3-6 分钟，aha₂ = 引用回答 + 图谱展开）
1. 引导气泡指向输入框："问一个只有你的资料能答的问题"——提供基于已摄入内容自动生成的 3 个建议问题（extract 角色生成）
2. 带引用回答完成 → spotlight 高亮引用角标："每句话都能溯源"→ 引导点击回填按钮 → 知识页生成动画 → **埋点 aha2_cited_answer**
3. 自动平移到图谱 tab → 宇宙视图首次展开（入场动画：节点从中心扩散）→ **埋点 aha2b_graph_reveal**

## 第三幕「派它干活」（6-9 分钟，aha₃ = 第一个 Agent）
1. 引导层："让 Terrane 替你盯一个话题"——预填模板：「每周一早上 8 点，搜集〈用户首库主题〉的新进展，写成周报放进库里」
2. 用户确认创建（cron 自动生成）→ 展示"下次运行时间"卡片 → **埋点 aha3_agent_scheduled**
- 无 web-search 渠道时：模板降级为"每周体检我的库并报告"（Lint 任务），文案如实说明

## 第四幕「它记得你」（次日回访，aha₄ = 记忆唤回）
1. 次日首次登录：Bell 推送记忆唤回（若有命中）+ 第一份 Lint 体检报告通知
2. 点开唤回通知 → spotlight 指向记忆面板入口："Terrane 在对话中自动记住了这些——你可以编辑或删除任何一条" → **埋点 aha4_memory_recall**
- 无唤回命中时：仅推 Lint 报告 + 记忆面板引导（aha4 记 partial）

## KPI 口径（PRD §2）
四 aha 触达率 = 各埋点 / 完成第一幕用户数；30 天留存以 product_events 活跃定义；漏斗在后台 Insights 页（v1 简版：四幕漏斗 + 留存曲线）。
